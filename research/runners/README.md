# Research runners

Stage scripts formerly at the repo root (`RUN_E5*` … `RUN_E10A*`, splash/UI demos).

They locate the monorepo by walking up to `CMakeLists.txt`.

```powershell
# from repo root
.\research\runners\RUN_E10A31R_A1B8C_PROVENANCE.ps1 -Seconds 90

# or the product-facing research aggregator
.\RUN_RESEARCH_GWY_SHELL.ps1
```

Screenshots go to `evidence/screenshots/` (not a root `screenshots/` folder).

**Not product.** Do not treat these as launch acceptance.
