目录结构  

1.sfw目录：SGL框架头文件
         Common 通用的函数模块
         Controls smp_控件模块
         Gui 核心模块
         Tsui 天使之翼开发的控件模块
         
2.lib目录的
         SGL_WIN32.lib          vs链接用的库，项目》属性》链接器
         sgllib_mtkt240.lib，   编译依赖的库

3.resource目录，  mrp开发用到的字图音表格，一键生成资源包与h文件，参照《Mythroad程序员开发指南（卷一：入门）》快速入门，游戏实战，游戏实现，字符读取



SGL新增加的东西：
1.屏幕缓冲区管理模块：screenbuf.c  使用的双缓冲技术，像listview scrollview 这样的控件
双缓冲可以使用拓展内存，或者主内存，编译开关 config.h ->USE_EXRAM
sgldef.h

2.TOAST模块：用于弹出即时消息

3.字库管理模块：font.c 封装了各种字库，绘制文本接口使用font.h提供的接口

由开发者去调用SGL框架函数，
其中这些函数又分别对应开发者的应用框架函数 ：
int InitApplication(VOID); 
int ExitApplication(VOID);
int PauseApplication(VOID);
int ResumeApplication(VOID);

event 函数没有

当 SGL_onInit 完毕时 会调用 InitApplication

int32 mrc_appEvent(int32 code, int32 param0, int32 param1)
{//计费状态下，自动返回
	SGL_onEvent(code, param0, param1);

	return MR_SUCCESS;
}

int32 mrc_appPause()
{//计费状态下，自动返回
	SGL_onPause();

	return MR_SUCCESS;	
}

int32 mrc_appResume()
{
	SGL_onResume();

	return MR_SUCCESS;
}
